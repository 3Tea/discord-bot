## Usage

| Command | Description |
|---------|-------------|
| `/profile` | View your own profile card |
| `/profile user:@someone` | View another user's profile |

## What's Shown

Your profile card includes:
- **Level & Rank** — current level, XP progress bar, server rank
- **Economy** — coins, gems, and stars
- **Streaks** — pray streak and quest streak
- **Activity** — total messages, voice time, reactions
- **Member since** — server join date

## Premium Tiers

| Tier | Format |
