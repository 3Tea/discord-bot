## Usage

```
/work
```

## How It Works

Use `/work` to earn a random coin reward between 80–200 coins (configurable by admins). Each work session comes with a fun flavor text describing your job.

There's a **4-hour cooldown** between work sessions (configurable). No options needed — just run the command and collect your pay.

> **Tip:** Combine `/work` with `/fish` and `/pray` for maximum daily income!

### Server Config
Admins can adjust cooldown, min/max reward via `/economy work-config-*` commands.
